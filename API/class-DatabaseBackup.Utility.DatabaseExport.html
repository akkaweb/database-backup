Class DatabaseExport

Utility to export the database.

Examples.

This will create a backup file with the gzip compression. The file will be created in the default directory, with a default name. In addition, only 10 backup files will be kept, the oldest will be deleted.

$backup = new DatabaseExport();
$backup->compression('gzip');
$backup->rotate(10);
$backup->export();

This will create the backup file mybackup.sql.bz2. in the directory /opt/backups. It will use the bzip2 compression (is automatically detected by the filename).

$backup = new DatabaseExport();
$backup->directory('/opt/backups');
$backup->filename('mybackup.sql.bz2');
$backup->export();
